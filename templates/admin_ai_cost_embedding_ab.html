Cutover Controls

When you're ready to flip production to the shadow model:

  1. Set MODEL_TIER_OVERRIDE_EMBEDDING_SERVICE_CANDIDATE=text-embedding-3-small in the environment.
  2. Workers pick it up on next restart (or rolling restart for zero-downtime).
  3. Leave EMBEDDING_AB_SHADOW_ENABLED=true for 7 days post-cutover so this page now monitors regression risk against text-embedding-3-large.

To revert: unset the override env var and restart. Job-embedding cache is per-model so old large-model vectors remain valid.
