How Tearsheet Selection Works

The system automatically loads all available tearsheets from your Bullhorn account. You can monitor single or multiple tearsheets:

  1. Make sure your Bullhorn API credentials are configured in Bullhorn Settings
  2. Single Tearsheet: Click on one tearsheet to select it
  3. Multiple Tearsheets: Hold Ctrl (or Cmd on Mac) and click on multiple tearsheets
  4. The system creates separate monitors for each selected tearsheet automatically
Pro Tip: When selecting multiple tearsheets, the system will create individual monitors named "Your Monitor Name - Tearsheet Name" for easy identification.

Note: If you don't see your tearsheet in the dropdown, check that your API credentials are correct and that you have access to the tearsheet in Bullhorn.
