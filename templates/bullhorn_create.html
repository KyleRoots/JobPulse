How Tearsheet Selection Works

The system automatically loads all available tearsheets from your Bullhorn account. Simply:

  1. Make sure your Bullhorn API credentials are configured in Bullhorn Settings
  2. Select the tearsheet you want to monitor from the dropdown menu
  3. The system will automatically handle the technical details behind the scenes

Note: If you don't see your tearsheet in the dropdown, check that your API credentials are correct and that you have access to the tearsheet in Bullhorn.
